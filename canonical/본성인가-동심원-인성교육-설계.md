# 本·性·人·家: 같은 가치를 자기에서 가족으로 넓히는 인성교육 설계

이 한문 인성교육 설계는 정직·배려 같은 덕목을 먼저 제시하지 않고 학생 자신에게서 출발한다. 1단계에서는 이름과 본관의 한자 뜻을 조사하며 가치를 받을 주체를 세운다(本). 2단계에서는 스스로 호(號)를 지어 자신의 지향을 언어로 굳힌다(性). 이어 자신이 세운 가치를 또래·학교폭력 예방 캠페인으로 외화하고(人), 마지막에 가족 토의로 끌어와 가훈으로 합의한다(家). 같은 가치를 자기 → 또래 → 가족으로 적용 대상만 넓혀 가며 반복하는 동심원 구조다.

원저자는 이 단계들의 모든 산출물을 하나의 [[성어 포트폴리오]]로 누적시킨다. 이름 성찰글, 호·전각, 가치관 에세이, 학폭예방 성어, 가훈 작품이 흩어지지 않고 한 묶음으로 쌓여, 학생이 자기 성장을 하나의 서사로 회수하도록 설계되어 있다.

이 설계를 실제 수업 사이클로 옮긴 운영 형태가 [[나선형-인성교육-구조]]이고, 효과 검증에서 무엇을 측정 대상으로 삼고 무엇을 제외하는지는 [[측정-경계-원칙]]에서 다룬다.
